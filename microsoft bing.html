About microsoft bing

Information about microsoft bing

Microsoft Bing is a web search engine owned and operated by Microsoft.The service has its origins in Microsoft's previous search engines MSN Search, Windows Live Search and later Live Search. Bing provides a variety of search services, including web, video, image and map search products.It is developed using ASP.NET.Bing, Microsoft's replacement for Live Search, was unveiled by Microsoft CEO Steve Ballmer on May 28, 2009, at the All Things Digital conference in San Diego, California, for release on June 3, 2009. Notable new features at the time included the listing of search suggestions while queries are entered and a list of related searches (called "Explore pane") based on semantic technology from Powerset,which Microsoft had acquired in 2008.In October 2011, Microsoft stated that they were working on new back-end search infrastructure with the goal of delivering faster and slightly more relevant search results for users Known as "Tiger".
